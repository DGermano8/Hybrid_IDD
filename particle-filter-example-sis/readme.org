#+title: SIS example using =pypfilt=

* Model

** Stoichiometric

|             | *Reactants* |     | *Products* |     | *Flow* |     |
|-------------+-------------+-----+------------+-----+--------+-----|
| *Rate*      |         $S$ | $I$ |        $S$ | $I$ |    $S$ | $I$ |
|-------------+-------------+-----+------------+-----+--------+-----|
| $\beta / N$ |           1 |   1 |          0 |   2 |     -1 |   1 |
| $\gamma$    |           0 |   1 |          1 |   0 |      1 |  -1 |

** Differential equations

\[
\frac{dS}{dt} = -\beta SI / N + \gamma I
\]

\[
\frac{dI}{dt} = \beta SI / N - \gamma I
\]
